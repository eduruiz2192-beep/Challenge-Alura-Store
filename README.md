# Desafío Alura Store: Análisis de Desempeño y Ventas

## Descripción del Proyecto
Este proyecto es un análisis de datos real simulado para *Alura Store. El objetivo principal es evaluar el rendimiento de cuatro sucursales diferentes y brindarle una recomendación estratégica al dueño (el Sr. Juan) sobre **qué tienda debería vender* para financiar un nuevo proyecto.

El análisis se basa en cinco pilares fundamentales:
1. Ingresos totales por tienda.
2. Categorías de productos más populares.
3. Calificación promedio de satisfacción del cliente.
4. Productos más y menos vendidos.
5. Costo de envío promedio.

(Incluye también un análisis geográfico extra de la distribución de las ventas).

##  Herramientas y Tecnologías Utilizadas
* *Lenguaje:* Python 
* *Manipulación y Análisis de Datos:* Pandas
* *Visualización de Datos:* Matplotlib
* *Entorno de Desarrollo:* Google Colab

## Conclusiones Principales
Tras procesar y visualizar los datos de las cuatro tiendas, se concluyó que *la Tienda 4 es la candidata ideal para la venta*. 
A pesar de mantener un alto nivel de satisfacción del cliente (4.0 estrellas) y ofrecer el costo de envío más bajo, es la sucursal que reporta los *menores ingresos totales*, lo que la convierte en el activo menos rentable para la empresa en este momento.
